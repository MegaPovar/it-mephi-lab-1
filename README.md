# Лабораторная работа №1

## Вариант 21.

Реализована полиморфная квадратная матрица. В качестве типов элементов используются:

- целые числа (`int`);
- комплексные числа (`Complex`).

Для матриц реализованы операции:

- сложение матриц;
- умножение матриц;
- умножение матрицы на скаляр.

## Как собрать и запустить

Для сборки используется `Makefile`.

```bash
make
```

После сборки появится исполняемый файл:

```bash
./matrix_lab
```

При запуске программа сначала выполняет встроенные тесты, а потом открывает консольное меню.

Очистить собранный бинарный файл можно командой:

```bash
make clean
```

## Что есть в меню

В программе можно:

- создать матрицу `A`;
- создать матрицу `B`;
- сложить `A + B`;
- умножить `A * B`;
- умножить матрицу `A` на скаляр;
- вывести `A`, `B` или результат;
- отдельно запустить встроенные тесты.

Для комплексных чисел ввод выполняется двумя числами: действительная и мнимая части.

Пример:

```text
1 2
```

означает комплексное число:

```text
1 + 2i
```

## Кратко про реализацию

Матрица хранится как один динамический массив:

```c
void* data;
```

Память под элементы выделяется через `calloc` в `matrix_init`.

Так как тип элемента заранее неизвестен, используется структура `FieldInfo`. В ней лежат указатели на функции для конкретного типа: сложение, умножение, умножение на скаляр, копирование, печать и получение нуля.

За счет этого одна и та же функция матрицы может работать и с `int`, и с `Complex`.

## Основные файлы

- `main.c` — консольное меню, ввод данных, запуск операций.
- `matrix.h`, `matrix.c` — структура матрицы и операции над матрицами.
- `field_info.h`, `field_info.c` — описание операций для разных типов элементов.
- `complex.h`, `complex.c` — структура и операции для комплексных чисел.
- `tests.h`, `tests.c` — встроенные тесты.
- `Makefile` — сборка проекта.

## Тесты

В тестах проверяются:

- создание матрицы с некорректным размером;
- сложение `int`-матриц;
- умножение `int`-матриц;
- умножение `int`-матрицы на скаляр;
- сложение комплексных матриц;
- умножение комплексных матриц;
- умножение комплексной матрицы `2x2`;
- умножение комплексной матрицы на комплексный скаляр;
- ошибка при скаляре неправильного типа;
- ошибка при сложении матриц разных размеров;
- ошибка при сложении матриц разных типов;
- обработка `NULL` вместо матрицы.

Текущий результат запуска тестов:

```text

Тестов провалено: 0
```
